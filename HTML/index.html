How does the web works?

All the systems that are connected to the web, are called as clients and servers. In simple terms, a client is a system which sends the request to the server and the server is the system, which will process that request and send a reply to the client in the form of response.
The moment you enter this address in your browser and you hit ENTER, a lot of different things happen:

  1. The URL gets resolved
  2. A Request is sent to the server of the website
  3. The response of the server is parsed
  4. The page is rendered and displayed
